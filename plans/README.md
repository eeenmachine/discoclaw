# DiscoClaw Plans

Integration plans that teach an AI agent how to extend DiscoClaw.
See `docs/discoclaw-plan-spec.md` for the format specification.

## Directories

| Directory | Purpose | Shipped? |
|-----------|---------|----------|
| `starters/` | Production-ready plans to get going fast | Yes |
| `examples/` | Illustrative plans that teach the format | Yes |
| `community/` | Your own plans (gitignored) | No |

## Creating a plan

Use the template at `templates/plans/integration.discoclaw-plan.md` or ask the
plan-generator skill to create one for you.
